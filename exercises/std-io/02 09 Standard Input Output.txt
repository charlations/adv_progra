 * Estandar de manejo de archivos en C
 * 	Es decir; se puede usar en cualquier versión de C en cualquier sistema operativo
 *	Un código escrito en Unix se puede usar en Windows o Mac sin ninguna modificación

 * Abre los archivos como TEXTO, *NO BINARIO*
 * 	Para abrir los archivos como binario:
 * 	|	file = fopen(filename, "rb"); 	<- b para leer en binario (como byte)
 * 	|	file = fopen(filename, "rw+b"); <- el w+ es para no truncar el archivo
 * 	|	file = fopen(filename, "rwb");	<- el w solo trunca si el archivo ya existe
